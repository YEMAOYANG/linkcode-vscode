# 编码风格规则

## 通用
- 使用 TypeScript strict 模式
- 优先使用函数式组件和 Hooks
- 变量命名使用 camelCase
- 组件命名使用 PascalCase

## 注释
- 复杂逻辑必须添加中文注释
- 函数使用 JSDoc 注释
- 不要添加无意义的注释
